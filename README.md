🌍 Universal AI Translator

A simple language translation tool built using **Python**, **Gradio**, and **Deep Translator**. It translates text into 100+ languages using Google Translate.

## Features

* Auto language detection
* Translate between 100+ languages
* Simple web interface
* Fast and easy to use

## Requirements

Install the required packages:

```bash
pip install gradio deep-translator
```

## Run the Program

```bash
python codealpha_task_1.py
```

After running, open the local Gradio link in your browser.

## Technologies Used

* Python
* Gradio
* Deep Translator (Google Translate API)
